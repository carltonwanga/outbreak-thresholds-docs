User Management

Creating a user

  1. Login to the application.
  2. Under user setup click Manage users.
  3. On the user management toolbar, click Add.
  4. Enter user details.
  5. Click Save.

When a user is added to the application the details will appear on the user management list and will receive a password reset email enabling the new user to set a password. The password reset link expires after 24 hours. If link has expired an administrator can resend a password reset link.

Deactivate Account

Deactivating a user account will prevent them from subsequently accessing the application. Account deactivation steps are as follows:

  1. Login as an administrator.
  2. Under User Setup, click Manage Users.
  3. Search and select a user (status must be active).
  4. On the user management toolbar, click Account Management dropdown menu.
  5. Click Deactivate.
  6. Click Yes on the confirmation dialogue.

Reseting User Password

An administrator is able to reset a user's password by following the steps below:

  1. Login as user with user management rights.
  2. Under User Setup, click Manage Users.
  3. Search and select a user.
  4. On the user management toolbar, click Account Management dropdown menu.
  5. Select Reset Password on the drop down sub-menu.
  6. Click Yes on the confirmation dialogue.

Users can also reset their own passwords by following the steps below If user enters valid email address, the user will receive a password reset link. The password reset link will prompt for a new password and reset the password. The User should be able to login with the new password.

Edit User

To edit user details

  1. Login as an administrator.
  2. Under user setup click Manage Users.
  3. Search and select a user.
  4. On the user management toolbar, click Edit
  5. Edit fields that you wish to save.
  6. Click save.

Search User

To search for a user follow the steps below

  1. Login as an administrator.
  2. Under user setup click manage users.
  3. On the user management toolbar, enter the user’s name or email or National ID/Passport number to search for a user.
  4. Click search.
